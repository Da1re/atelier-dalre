# @atelier/web

Atelier Dalre 포트폴리오 사이트 — Next.js 16 App Router 기반.

> 전체 프로젝트 개요·구조·스크립트는 루트 [`README.md`](../../README.md)를 참고하세요.

## Quick Start

```bash
# 루트에서 한 번만 설치
pnpm install

# 이 앱만 개발 서버 실행
pnpm --filter @atelier/web dev
# → http://localhost:3000
```

## Scripts

| Command | Description |
| --- | --- |
| `pnpm dev` | Next.js dev server |
| `pnpm build` | Production build |
| `pnpm start` | Production server |
| `pnpm lint` | ESLint |
